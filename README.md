# 算法竞赛学习笔记

LaTeX 编写的算法竞赛知识体系，涵盖常见算法与数据结构的理论讲解和代码模板。

## 目录

| 专题 | 说明 |
|------|------|
| 基础算法 | 排序、二分、前缀和与差分、双指针等 |
| DP | 背包、线性DP、区间DP、状压DP等 |
| 图论 | 最短路、最小生成树、拓扑排序、连通性等 |
| 数据结构 | 线段树、树状数组、平衡树、分块等 |
| 搜索 | DFS、BFS、剪枝、A*等 |
| 字符串 | KMP、Trie、AC自动机、后缀数组等 |
| 计算几何 | 点积叉积、凸包、半平面交等 |
| 数学思维 | 数论、组合数学、博弈论等 |
| 并查集 | 基础并查集与带权并查集 |
| 模板代码 | 常用代码模板汇总 |
| ICPC 考点 | ICPC 竞赛考点大纲 |
| 算法总览 | 各专题整合总集 |

## 使用

每个目录包含 `.tex` 源文件和编译好的 `.pdf`，可直接下载 PDF 阅读。
